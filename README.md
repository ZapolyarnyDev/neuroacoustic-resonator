# neuroacoustic-resonator

Neuroacoustic Metabolic Resonator - экспериментальная самоорганизующаяся динамическая среда, которая преобразует живой звук в эволюционирующие структуры фазового поля и генерирует ответный звук из возникающих градиентов синхронизации

**Главная фишка - отсутствие начальной информации и словарей** 

Среда не является языковой моделью или чат-ботом, она представляет собой множество осцилляторов, поток энергии, адаптацию к информации из аудио-входа и генерацию исходящего звука

Проект исследует, могут ли устойчивые акустические паттерны возникать в двумерном поле связанных фазовых осцилляторов с локальным метаболизмом, пластичностью, предиктивной ошибкой, аудио-входом и аудио-выходом

## Текущий статус

На данный момент запланированы живые карты фазы и синхронности на основе динамики осцилляторов, без микрофона и аудио

## Стек

- Python 3.14
- uv для управления зависимостями
- NumPy для работы с полем осцилляторов в виде массивов
- SciPy для обработки сигналов и численных утилит
- Numba для ускорения обновлений поля
- sounddevice для аудио-входа и аудио-выхода
- PySide6 и pyqtgraph для визуализации состояния среды
- Pydantic и PyYAML для конфигурации проекта
- tqdm для отображения прогресса экспериментов

### Инструменты разработки

- Ruff - линтинг
- Mypy - статическая проверка типов
- Pytest - для тестов
