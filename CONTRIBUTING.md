# Участие в проекте

Проект находится на ранней стадии (провожу исследования из личного интереса). До первой полноценной версии пулл реквесты не принимаются, но вопросы, идеи и наблюдения можно оставлять в issues :3

## Локальный запуск

Требуется Python 3.13 и uv

Установить зависимости:

```sh
uv sync --dev
```

Запустить проект:

```sh
uv run python main.py
```

Проверить код:

```sh
just check
```

Запустить pre-commit по всем файлам:

```sh
just hooks
```

## NixOS

Войти в окружение:

```sh
nix develop
```

Если установлен direnv:

```sh
direnv allow
```
